# velocikeys-privacy

Public-facing privacy policy and data-safety mapping for the
[VelociKeys AI](https://github.com/vivaxlogic/velocikeys-ai) Android
keyboard.

This repo exists to host a stable public URL that the Google Play Store
listing can link to — it is **not** the application source. The full
project lives in the private `velocikeys-ai` repo.

## Live URL

📄 **<https://vivaxlogic.github.io/velocikeys-privacy/>**

## Contents

| File | Purpose |
|------|---------|
| [`privacy.md`](privacy.md) | The privacy policy submitted to Google Play. |
| [`data-safety.md`](data-safety.md) | The exact mapping for every Play Console Data Safety question. |
| [`index.md`](index.md) | Landing page that links to both documents from GitHub Pages. |

## Updating

When the privacy policy needs to change:

1. Edit the canonical version in
   [`vivaxlogic/velocikeys-ai/docs/PRIVACY_POLICY.md`](https://github.com/vivaxlogic/velocikeys-ai/blob/main/docs/PRIVACY_POLICY.md).
2. Copy the updated file to this repo as `privacy.md`.
3. Push — GitHub Pages rebuilds within ~1 min.
4. Update the "Last updated" date inside the document body.

The same flow applies to `data-safety.md`.
